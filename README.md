# get-skole-aar

Returns school year

## Installation

```sh
$ npm i get-school-year
```

## Usage

```JavaScript
const getSchoolYear = require('get-school-year')

getSchoolYear('2020-08-15') //=> 2020/2021
getSchoolYear('2020-02-01') //=> 2019/2020
getSchoolYear('2019-04-01') //=> 2018/2019
```
